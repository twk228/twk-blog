# twk-blog

## 🚀 简介
唐伟科的个人技术博客，这个项目用于记录个人的项目经验、做过的一些练习、还有个人经常的使用导航。

## 项目架构
React，webpack


## 🎥 预览
- [twk-blog](https://twk-blog.netlify.app)

## 🏗️ 使用

- node版本 >= 18.0.0（推荐20.6.0）

- 获取项目代码

```bash
git clone https://github.com/twk228/twk-blog.git
```

- 安装依赖

```bash
npm i

```

- 运行

```bash
npm start
```

- 打包

```bash
npm run build
```
